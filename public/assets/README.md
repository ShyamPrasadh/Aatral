# Assets Directory

This directory contains all static assets for the AATRAL Energy Dashboard.

## Structure

- **images/**: Place all PNG and JPEG images here
- **icons/**: Place all SVG icons here

## Usage

To use assets in your components:

```tsx
// Images
<img src="/assets/images/logo.png" alt="AATRAL Logo" />

// Icons
<img src="/assets/icons/energy.svg" alt="Energy Icon" />
```

## Recommended Image Formats

- **PNG**: For images with transparency
- **JPEG**: For photographs and complex images
- **SVG**: For icons and simple graphics (scalable)

## Optimization Tips

1. Compress images before adding them
2. Use appropriate dimensions (don't upload oversized images)
3. Use SVG for icons when possible for better scalability
4. Consider using WebP format for better compression
